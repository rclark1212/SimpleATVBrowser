Simple Browser for Android TV Help

This is a very simple browser example which allows the loading of web pages under Android TV. It has a few optimizations specifically for leanback mode including voice entry of web addresses as well as a favorites starting page for easy selection of your commonly accessed web pages.

Controller Usage:

In general, all the standard controller buttons work as you expect. Home and back will take you out of the application. Voice search gets overriden to allow entry of a URL address. Some functions will only work when on main web page.

Main Page:

When you launch the application for the first time, you will be taken to the main page. There is a search bar at top with controls and address entry with a web page below.

Help - pulls up this screen

Favorite Screen - toggles between this screen and your favorites screen (must have at least one favorite set)

Favorite Tag - toggles the current web site to be a favorite or not - if marked as favorite (heart is solid), site will show up on the favorites screen (read about favorites screen below)

Back/Forward - navigate through your web history

Voice Entry - toggles voice entry for web address entry (voice entry tool will pop up and will transcribe your voice to address bar)

Refresh - reloads current page

List of Last Address - pulls up a dropdown list containing a selectable list of your last visited web pages

Address Entry - enter your web page address here.

Important - the logic used for processing text entry into the address bar is as follows:

  • If text begins with http:// or https://, app attempts to load that web site regardless
  • If text is “info”, it will pull up this help screen
  • If text is determined to be a valid web url via pattern matching, app will attempt to load it
  • If text is not a valid url, app will do a google search on the text entered

Favorites Page:

The favorites page offers a very simple scrollable view of web sites that you have marked as favorites. The app will automatically launch to this page if one or more favorites is set. You can toggle between this view and the main page by clicking the home button in the search bar.

The web page thumbnails are generated at the time that you mark them as a favorite (they are not updated live). To update the thumbnail view, click on the web site and then uncheck and recheck the favorite tag icon in the search bar.

To remove a favorite, simple launch that web page and uncheck the favorite tag icon.

About

This application has only been tested on a NVIDIA Shield TV. It should work on other Android TV platforms although you will not get right-stick mouse functionality.

This application uses the android webview class at its core. Javascript and cookies have been enabled. However there has not been any security review, tabs are not supported, nor has any caching been enabled. And the android webview class does not support plugins which mean certain websites that require plugin support will not work (for example, hangouts).

Sources for this project are available on github at https://github.com/rclark1212/SimpleATVBrowser.
